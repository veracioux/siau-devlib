===========
SIAU devlib
===========

.. rubric:: Biblioteka za čitanje uređaja iz JSON datoteka

Biblioteka *devlib* se koristi za parsiranje JSON datoteka koje sadrže opise
uređaja, kako bi se dobila C++ reprezentacija uređaja. Ove funkcije su izdvojene
u zasebnu biblioteku da bi ih drugi timovi mogli koristiti za svoje potrebe.
